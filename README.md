Features
- Dynamic Activity Dashboard
Toggle between Day / Week / Month views with smooth transitions.
- Bar Charts for Activity Metrics
Visualize calories, steps, and heart rate using custom, lightweight chart components.
Beautifully styled cards showing workouts, durations, and progress.
- Modern UI 
Clean typography, soft gradients, and intuitive layout.
- Modular Architecture
Components, navigation, and theme separated for maintainability and scalability.
- Fast Iteration with Expo
Easy development, testing, and deployment.

Why I Built This
I wanted a fitness dashboard that feels modern, smooth, and professionally designed — but fully customizable and open source.
This project is also a playground for experimenting with:
- modular UI architecture
- custom chart components
- clean, reusable design patterns
