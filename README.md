# Sample Snack app

This project is my sport‑tracking app built with Expo.
It tracks randomly steps, updates kcal based on movement, and lets you reset all data and settings whenever you want.

The app shows charts for RAI, TSS, Running Power and other metrics. These values are calculated from your GPS‑based activities or from activities you enter manually. You can also browse your full activity history, and the homepage always displays your most recent activity.

FTP is not measured automatically — you need to enter your own value manually.
You can switch between kg and lbs by tapping the “kg” label. When the unit changes, distance automatically switches between kilometers and miles.

The app is still in development, but the main features are already working.
More improvements, better UI and additional metrics will be added over time.
